# Playlist Maker
Учебный проект с курса "Android разработка" Яндекс Практикума. Приложение для позволяет искать треки в базе iTunes, прослушивать 30 секундный отрезок и создавать плейлисты. 

## Используемые технологии и инструменты
- Kotlin
- Room
- Retrofit
- Kotlin Coroutines
- Koin
- ViewBinding
- RecyclerView
- Glide
- ViewPager
- SharedPreferences
- Clean Architecture (MVVM)

## Функционал приложения
- Поиск песен в базе iTunes
- Прослушивание 30 секундного ознакомительного отрывка песни
- История поиска (10 последних треков)
- Добавление/удаление треков в избранное
- Создание плейлистов
- Удаление треков из плейлиста
- Редактирование информации о плейлисте
